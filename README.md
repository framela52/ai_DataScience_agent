# AI Agent DS Analyst

AI-агент аналитик данных на базе GigaChat с ReAct рассуждениями, веб-поиском, анализом корреляции и метриками качества.

## Архитектура проекта

ai_ds_agent/
  agent/
    main_agent.py               # GigaChatAgent - основной класс агента

  monitoring/
    evaluator.py                # GigaChatJudge - оценка качества (полнота, точность, ясность)
    metrics.py                  # MetricsCollector - перплексия, релевантность, задержка

  tools/
    data_analysis.py            # analyze_correlation() - корреляция Пирсона
    gigachat_llm.py             # GigaChatLLM - низкоуровневый клиент API
    web_search.py               # web_search() - поиск через DuckDuckGo

  app.py                        # HTTP сервер + ThinkingAgent + веб-интерфейс
  .env                          # GIGACHAT_CLIENT_ID, GIGACHAT_CLIENT_SECRET
  requirements.txt              # Зависимости Python

## Компоненты системы

### 1. Агент (`agent/main_agent.py`)

Класс `GigaChatAgent` - основной агент для общения с GigaChat API.

**Особенности:**
- Аутентификация через Client ID / Client Secret
- Отправка запросов к модели GigaChat
- Очистка ответов

**Методы:**
- `run(query)` - отправка запроса и получение ответа

### 2. Клиент GigaChat (`tools/gigachat_llm.py`)

Класс `GigaChatLLM` - низкоуровневый клиент для работы с API GigaChat.

**Функции:**
- Аутентификация и получение токена
- Отправка запросов к API

### 3. Веб-поиск (`tools/web_search.py`)

Функция `web_search()` - поиск информации в интернете.

**Особенности:**
- Использует DuckDuckGo (бесплатно, без API ключа)
- Возвращает до 3 результатов с заголовками и ссылками

### 4. Анализ корреляции (`tools/data_analysis.py`)

Функция `analyze_correlation()` - расчет коэффициента корреляции Пирсона.

**Формат входа:** `x_переменная,y_переменная,значение1,значение2,...`

### 5. Оценка качества (`monitoring/evaluator.py`)

Класс `GigaChatJudge` - оценка ответов по трем критериям.

**Критерии:**
- Полнота (0-1)
- Точность (0-1)
- Ясность (0-1)

### 6. Сбор метрик (`monitoring/metrics.py`)

Класс `MetricsCollector` - сбор и агрегация метрик производительности.

**Метрики:**
- Перплексия (связность текста)
- Релевантность (ответ на вопрос)
- Задержка ответа
- Длина ответа
- Полнота, точность, ясность (через GigaChatJudge)

В качестве LLM судьи была выбрана та же модель, так как она бесплатная, не требует локальныйх ресурсов и работает быстро с русским языком.

### 7. Веб-интерфейс (`app.py`)

HTTP сервер на базе `http.server` с двумя эндпоинтами:

| Эндпоинт | Метод | Описание |
|----------|-------|----------|
| `/` | GET | Главная страница с чатом |
| `/api/ask` | POST | Отправка запроса агенту |

**Интерфейс:**
- Левая панель - процесс рассуждения агента. В ней можно увидеть, какие интсрументы использует агент и источники, откуда он берет информацию.
- Правая панель - финальный ответ
- Нижняя панель - метрики и статистика

## Установка и запуск

### 1. Клонирование и установка зависимостей

# Клонируйте проект
cd ai_ds_agent

# Создайте виртуальное окружение
python -m venv venv_ds

venv_ds\Scripts\activate  # Windows
source venv_ds/bin/activate  # Linux/Mac

# Установите зависимости
pip install -r requirements.txt

# Запуск
python app.py

Откройте браузер: http://localhost:8000
